# Void-Editor SilenceCut

Free Local AI-powered silence remover plugin for Adobe Premiere Pro.

## Features
-  Automatic silence detection
-  One-click silence removal
-  Waveform preview with zoom & pan
-  Auto threshold detection
-  Built-in presets (Podcast, Interview, Lecture, Vlog)
-  Custom preset save/load
-  Multiple edit modes (Linked, Audio Only, All Tracks)
-  100% Offline — No internet required

## Installation
1. Download the latest installer from [Releases](../../releases)
2. Run `VoidEditor_SilenceCut_Setup_v1.0.0.exe`
3. Open Premiere Pro
4. Go to `Window → Extensions → Void-Editor SilenceCut`

## System Requirements
- Windows 10/11 (64-bit)
- Adobe Premiere Pro 2024 or later
